# Pseudo Classes

Pseudo class digunakan untuk menambah efek special pada selector-selector tertentu.
Pada CSS Pseudo Class bisa dikombinasikan dengan class, kompatible dengan browser.
Class berjalan dengan baik, pastikan 
```
<!Doctype HTML>
```
Dimasukan pada baris kode paling atas, berikut selector pseudo class:

| Struktur | Contoh | Penjelasan |
| :--- | :--- | :--- |
| **[:link]** | a:link | Link yang belum dikunjungi. |

| **[04. Manajemen Data](chapters/04-eloquent-dan-manajemen-data.md)** | Eloquent | Mass Assignment, Accessors, Transactions. |

| **[06. Presentasi](chapters/06-presentasi-dan-frontend.md)** | Output | API Resources, Blade Clean Code, Assets. |
